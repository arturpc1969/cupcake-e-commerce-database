# cupcake-e-commerce-database
Banco de dados para o projeto e-commerce de cupcakes

## Instruções:

Para executar localmente o servidor de banco de dados Postgres do projeto, execute a linha de comando abaixo na raiz deste repositório:

```bash
docker-compose up -d
```

Para finalizar a execução do servidor, execute a linha de comando abaixo na raiz deste repositório:

```bash
docker-compose down
```
